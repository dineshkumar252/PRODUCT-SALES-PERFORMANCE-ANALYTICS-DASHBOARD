 Top 5 Mobile Brands Sales Analytics Dashboard
Project Overview
The Top 5 Mobile Brands Sales Analytics Dashboard is a Business Intelligence project developed using Power BI to analyze and visualize the sales performance of leading smartphone brands. The project uses a structured sales dataset to provide meaningful insights into revenue generation, sales volume, regional performance, and customer purchasing trends through interactive dashboards and data visualizations.
The primary objective of this project is to help business stakeholders identify the best-performing mobile brands, monitor sales trends, compare regional performance, and support data-driven decision-making. By transforming raw sales data into interactive reports, the dashboard enables users to quickly evaluate key performance indicators (KPIs) and uncover actionable business insights.
The dashboard incorporates data modeling, Power Query for data preparation, and DAX (Data Analysis Expressions) to calculate business metrics such as Total Revenue, Total Units Sold, Average Selling Price, and Total Sales Transactions. Interactive visualizations, including KPI cards, bar charts, line charts, pie charts, tables, and slicers, allow users to explore sales performance across different brands, regions, models, and time periods.
Project Objectives
•	Analyze the sales performance of the top five mobile brands.
•	Identify the highest-selling brands based on revenue and units sold.
•	Compare sales performance across different geographical regions.
•	Track monthly sales and revenue trends.
•	Evaluate the performance of individual mobile models.
•	Provide an interactive dashboard for business analysis and reporting.
Key Features
•	Interactive KPI cards displaying Total Revenue, Total Units Sold, Average Selling Price, and Total Sales Transactions.
•	Top 5 mobile brands analysis using dynamic bar charts.
•	Monthly sales trend analysis with line charts.
•	Regional sales distribution using pie or donut charts.
•	Brand-wise units sold comparison.
•	Top-selling mobile models displayed in a detailed table.
•	Interactive slicers for Brand, Region, and Date to enable dynamic filtering.
•	Custom DAX measures for accurate business performance calculations.
Tools & Technologies
•	Microsoft Power BI
•	Microsoft Excel
•	Power Query
•	DAX (Data Analysis Expressions)
•	Data Modeling
Business Value
This project demonstrates how Power BI can transform raw sales data into meaningful business insights. It enables organizations to monitor key performance metrics, identify market trends, evaluate product performance, optimize inventory planning, and support strategic business decisions through interactive dashboards.
This project showcases practical skills in data visualization, dashboard development, data modeling, DAX, and business analytics, making it suitable for Business Analyst, Power BI Developer, and Data Analyst portfolios.
